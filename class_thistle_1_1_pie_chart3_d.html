Detailed Description

The Thistle::PieChart3D class provides a default implementation for 3D-piecharts.

The Thistle::PieChart3D class considers that each cell int he first column from the model is a serie of values. It represents a serie with a slice in a disk. The vertical headers defines the name of data.
