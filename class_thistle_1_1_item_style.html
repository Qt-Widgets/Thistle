Detailed Description

Thistle::ItemStyle class is used by Thistle::ItemDelegate to set the look and feel of item.

An item style consists of :

  • a QBrush for background
  • a QPen for the border
  • a QFont for the text
  • a Thistle::Shape for the item shape
